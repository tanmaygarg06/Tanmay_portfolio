## 💡 About This Project
This portfolio is built with a focus on **premium aesthetics, high-performance animations, and zero-dependency HTML/CSS**. It utilizes sleek glassmorphism, completely custom CSS cubic-bezier hover animations, and a dynamic HTML5 Canvas background to immediately catch the eye.

### Features
- **Responsive Design**: Fluid and accessible layouts across all desktop and mobile devices.
- **Serverless Form Integration**: Fully functional contact form routing directly to Gmail using `FormSubmit` via native POST requests, complete with automated "Thank You" replies.
- **Dynamic Theming**: Integrated Light/Dark mode featuring separate, fully-customized aesthetic paradigms (radial-geometric versus deep-space canvas).
- **Interactive Animations**: IntersectionObserver reveals, custom typing effects, and custom mouse cursors.

## 🛠️ Technology Stack
- **HTML5**: Semantic and accessible document structure.
- **CSS3**: Variables, Flexbox/Grid, and advanced pure-CSS animations (No external heavy libraries like Bootstrap or Tailwind used).
- **Vanilla JavaScript**: All DOM manipulation, intersection observation, typing effects, and canvas animations are written cleanly from scratch.

## ⚙️ Running Locally
1. Clone this repository: 
   ```bash
   git clone https://github.com/tanmaygarg06/portfolio.git
   ```
2. Open the directory.
3. Open `index.html` in your favorite web browser!

*Note: For the Contact Form and its auto-responder to function legitimately without throwing a security error, the files must be hosted on a web server (e.g., GitHub Pages, Vercel, or a local python/live-server).*
